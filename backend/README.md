# BSI Telemetry - Backend

Node.js/Express backend API for the BSI Telemetry Reporting System.

## Quick Start

```bash
npm install
npm start        # Production
npm run dev      # Development with hot-reload
```

## Full Documentation

## API Base URL

```text
http://localhost:5000/api
```

## Tech Stack

- Node.js 22
- Express 5
- MySQL 8.0
- JWT Authentication
