# SSL/TLS Setup Guide

Complete SSL setup documentation is available in the **[Detailed Guide](../DETAILED_GUIDE.md#ssltls-setup)**.

## SSL Options Covered

1. **Let's Encrypt** (Free, recommended) - Automatic renewal
2. **Self-Signed** - Development/testing only
3. **Commercial Certificate** - DigiCert, GlobalSign, etc.

See the **[Detailed Guide](../DETAILED_GUIDE.md#ssltls-setup)** for step-by-step instructions.
